# docker-WordPress-practice
practice for docker


## command

### docker

#### docker initialize
```shell script
$ docker-compose up -d
```
```shell script
$ docker exec -it wp1-wordpress /bin/bash
```
```shell script
$ chmod +x /tmp/wp-install.sh
```
```shell script
$ /tmp/wp-install.sh
```

#### docker start
```shell script
$ docker-compose up -d
```
#### docker end
```shell script
$ docker-compose down
```
